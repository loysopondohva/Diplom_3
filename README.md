## Дипломный проект. Задание 3: веб-приложение

### Автотесты для проверки веб-приложения Stellar Burgers.

### Реализованные сценарии

#### Проверка основного функционала (`test_main_page.py`)
- Проверка перехода по клику на «Конструктор»;
- Проверка перехода по клику на «Лента заказов»;
- Проверка если кликнуть на ингредиент, появится всплывающее окно с деталями;
- Всплывающее окно закрывается кликом по крестику;
- при добавлении ингредиента в заказ счётчик этого ингредиента увеличивается.
залогиненный пользователь может оформить заказ.

#### Раздел «Лента заказов» (`test_order_feed_page.py`)
- Проверка при создании нового заказа счётчик «Выполнено за всё время» увеличивается;
- Проверка при создании нового заказа счётчик «Выполнено за сегодня» увеличивается;
- Проверка после оформления заказа его номер появляется в разделе «В работе».

## Структура проекта

- `data.py` - содержит данные для тестов.
- `conftest.py` - настройки и фикстуры для тестов.
- `urls.py` - URL адреса, для тестов.
- `locators/account_page_locators.py` - локаторы для страницы пользователя и авторизации.
- `locators/main_page_locators.py` - локаторы для основной страницы, и создания бургеров.
- `locators/order_feed_page_locators.py` - локаторы для Ленты Заказов.
- `pages/base_page.py` - POM, основные методы для описания действий на страницах.
- `pages/account_page.py` - POM, основные методы для описания действий на странице аккаунта пользователя и логинации.
- `pages/main_page.py` - POM, основные методы для описания действий основной странице/конструкторе.
- `pages/order_feed_page.py` - POM, основные методы для описания действий на странице "Лента Заказов".
- `tests/test_main_page.py` - Тесты основной страницы.
- `tests/order_feed_page.py` - Тесты страницы "Лента Заказов".
- `allure-report-chrome/index.html` - allure отчёт о прогоне тестов в Chrome.
- `allure-report-firefox/index.html` - allure отчёт о прогоне тестов в Firefox.

### Запуск автотестов

Для Chrome (по умолчанию):
pytest

Для Firefox:
pytest --browser=firefox

## Инструкция по запуску:

### 1. Установите зависимости:

> pip install -r requirements.txt

### 2. Запустить все тесты и записать отчет:

Для Chrome (по умолчанию):
> pytest --alluredir=./allure-results

Для Firefox:
> pytest --browser=firefox --alluredir=./allure-results-firefox

### 3. Посмотреть отчет по прогону html

> allure serve ./allure-results

### 4. Сгенерировать отчёт в один файл, по прогону html
> allure generate --single-file ./allure-results
